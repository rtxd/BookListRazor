## Book List Application

This application is a small book list application that allows you to add, edit and delete to a list of books. It uses a local MSSQL Database to store the data.

I made this application to improve my understanding of C# Asp.Net applications, I followed a tutorial that can be found [here](https://www.youtube.com/watch?v=C5cnZ-gZy2I&t=8333s)

## Tech/framework used

<b>Built with</b>

- ASP.NET Core 3.1
- Razor Pages
- MSSQL
